Project 3a: Image Warping and Mosaicing

Overview

The goal of this project was to explore image warping and how to apply it to create an image mosaic. A mosaic is formed by aligning and blending multiple images of the same scene with a fixed center of projection (the camera rotates but does not move), each with different perspectives, into a seamless panorama.

Warping is achieved by selecting corresponding points on two images that represent the same real-world locations, then computing a homography — a 3×3 matrix that maps every point in one image to the corresponding point in another.

Warp the Images

To warp images using a homography, I used inverse warping to avoid holes in the output. Each pixel in the output is mapped back to its source position using H⁻¹. I implemented both Bilinear and Nearest Neighbor interpolation methods:

  • Nearest Neighbor: Copies the value from the closest pixel center (fast but less smooth).
  • Bilinear: Averages the four surrounding pixels weighted by proximity (smoother, better for mosaics).

Blend the Images into a Mosaic

I aligned and blended the warped images using alpha masks and Laplacian stacks to create smooth transitions. Each image contributes more strongly where its alpha mask is higher, with distance transforms ensuring gradual fading near overlaps.
